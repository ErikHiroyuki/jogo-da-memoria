##PROJETO: JOGO DA MEMÓRIA

##O que é jogo da memória?

O jogo da memória é um clássico jogo formado por peças que apresentam uma figura em um dos lados. Cada figura se repete em duas peças diferentes. Para começar o jogo, as peças são postas com as figuras voltadas para baixo, para que não possam ser vistas. 

##- OBJETIVO DO JOGO
Encontrar todos os pares em menor tempo e momvimentos posssível.

##- COMO JOGAR 
O Jogador deve clicar sobre duas cartas para forma um par, caso não formem um par, a carta e virada para a mesa. O jogo continuará aé que formem todos os pares.
